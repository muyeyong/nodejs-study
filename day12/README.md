#### 模板引擎使用
 在使用模板引擎的时候
 vm.runInContext(
  `(function (data) {
    with (data) {
        return ${data}
    }
})`, vm.createContext({})
 会提示data没有定义，在runInContext第一个参数不能传递没有定义的值(这个值必须能够在当前上下文可以访问，不能是以后传入的值)
